# HeCopUI Framework 

HeCopUI is a premium, high-performance UI components library for Windows Forms (.NET Framework), designed to help developers create modern, professional, and visually stunning desktop applications with ease.

![image](https://github.com/Khanh779/HeCopUI_Framework/blob/master/Screenshot/Record1.gif)

## Core Features

- **Modern Aesthetics:** Clean, premium designs inspired by modern UI trends (Teal/DodgerBlue accent system).
- **High Performance:** Optimized rendering using GDI+ with double buffering and advanced smoothing techniques.
- **Rich Component Library:** 80+ custom controls and components ready for production.
- **Dynamic Animations:** Built-in animation engine for smooth transitions and interactive micro-animations.
- **Visual Effects:** Native support for Gaussian Blur, Box Blur, and dynamic shadows.
- **Designer Friendly:** Full support for Visual Studio Property Grid and Drag-and-Drop.

## Built-in Effects & Helpers

- **Effects:** Gaussian Blur, Box Blur, Shadow Bitmap, Slide Effects, Color Light Effects.
- **Animations:** `AnimationManager`, `Animator`, `Controller`, `DecorationControl`, `DoubleBitmapControl`.
- **Helpers:** `ColorHelper`, `DrawHelper`, `GraphicsHelper`, `TextHelper`, `DPIHelper`, `HBlurs` for advanced drawing operations.

## Getting Started

### Prerequisites

- .NET Framework 4.7.2 or higher.
- Visual Studio 2019/2022.

### Installation

Clone the repository and add the `HeCopUI_Framework` project to your solution:

```bash
git clone https://github.com/Khanh779-9/HeCopUI_Framework.git
```
